# Real-Time Chat Application

## Objective
To build a real-time chat application using Node.js, Express, and Socket.IO.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Socket.IO

## Features
- Real-time messaging
- Frontend and backend integration
- User-friendly interface
- Multi-user communication

## How to Run

1. npm install
2. node server.js
3. Open http://localhost:3000

## Outcome
Successfully developed a real-time chat application that allows instant communication between users.
